Introduction

Welcome to the HyperMenu documentation. This guide will walk you through the installation, configuration, and optimization of the menu framework. HyperMenu is designed to be lightweight, efficient, and fully customizable.

Notice: HyperMenu 3.0 is now the standard stable version. Earlier versions are deprecated.

Quick Setup

To initialize the core menu system, paste the following command into your project terminal:

# Initialize HyperMenu via Bash
curl -sSL https://install.hypermenu.io | sudo bash

After installation, the system will generate a default configuration file in your root directory.

Default Keybinds

Config Format

Your config.json should follow the standard schema to ensure compatibility with the rendering engine:

{
  "theme": "modern-blue",
  "opacity": 0.95,
  "animations": true,
  "settings": {
    "speed": "ultra",
    "debug_mode": false
  }
}
