01. Installation

To get started with HyperMenu, you need to initialize the core script within your project directory. Run the following command in your terminal to begin the automated setup.

Bash
curl -sSL https://hypermenu.io/install.sh | bash

02. Configuration

Edit your config.json to match your preferred theme and keybinds. Below is the standard schema: 

{
    "version": "3.0.0",
    "settings": {
        "theme": "neon-dark",
        "autoUpdate": true,
        "latencyMode": "ultra"
    }
}

03. Control Mapping

Use these default keys to navigate the HyperMenu interface.

Action Keybind Description
Open Menu F4 Toggles the main dashboard overlay.
Quick Select ENTER Confirms the highlighted option.
Back/Exit ESC Exits the current menu view.
