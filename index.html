How It All Started

Back in 2024, everything was slow. We spent months stripping away the bloat and rewriting the core engine to prioritize pure speed. Today, HyperMenu is the fastest choice for developers worldwide.

Documentation

Everything you need to set up HyperMenu in seconds.

1. Installation

Run this command to install the HyperMenu core directly into your environment:

$ git clone https://github.com/HyperMenu/Project.git
$ cd HyperMenu
$ npm install && npm start

2. Configuration

Update your settings.json to enable high-performance mode:

{
  "theme": "neon-glass",
  "speed": "ultra",
  "lowLatency": true,
  "keybind": "F4"
}

3. Control Keybinds

Function Key Action
Toggle Menu F4 Opens/Closes the overlay
Select Item Enter Confirm selection
Navigation Arrows Move between options
