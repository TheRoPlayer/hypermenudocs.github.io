1️⃣ Requiring the Loader in a script In a LocalScript: local ReplicatedStorage = game:GetService("ReplicatedStorage") local Loader = require(ReplicatedStorage.Loader) Now you can use everything from the library. 2️⃣ Creating the UI window local Window = Loader:CreateWindow({ Name = "Boys VS Girls | Hyper UI", Subtitle = "Custom UI Library", Width = 850, Height = 550, Accent = Color3.fromRGB(170,85,255) }) Now Window controls the UI. 3️⃣ Creating tabs local Main = Window:CreateTab("Main") local Movement = Window:CreateTab("Movement") local Settings = Window:CreateTab("Settings") 4️⃣ Adding a button Main:CreateButton({ Name = "Print Hello", Callback = function() print("Hello world") end }) 5️⃣ Adding a toggle Movement:CreateToggle({ Name = "Fly", Default = false, Callback = function(state) print("Fly:", state) end }) state will be true or false. 6️⃣ Adding a slider Movement:CreateSlider({ Name = "WalkSpeed", Min = 16, Max = 100, Default = 16, Callback = function(value) print("Walkspeed:", value) end }) value is the number. 7️⃣ Adding a dropdown Settings:CreateDropdown({ Name = "Choose Team", Options = {"Red","Blue","Green"}, Callback = function(option) print("Selected:", option) end }) 8️⃣ Adding an input box Settings:CreateInput({ Name = "Say Something", Placeholder = "Type text here...", Callback = function(text) print("You typed:", text) end }) 9️⃣ Sending a notification Window:Notify({ Title = "Hello", Content = "This is a notification", Duration = 3 }) 🔟 Changing the credits Window:SetCredits( "My Credits", "UI by Ben\nLibrary by Hyper UI" ) 11️⃣ Changing the accent color Window:SetAccent(Color3.fromRGB(0,170,255)) 12️⃣ Changing the footer text Window:SetFooter("UI Loaded Successfully") 13️⃣ Key system configuration (before creating window) Loader.Config.KeySystem.Enabled = true Loader.Config.KeySystem.Keys = {"BEN123"} or disable: Loader.Config.KeySystem.Enabled = false 🧠 Example full script Example complete usage: local ReplicatedStorage = game:GetService("ReplicatedStorage") local Loader = require(ReplicatedStorage.Loader) Loader.Config.KeySystem.Enabled = true Loader.Config.KeySystem.Keys = {"HELLO123"} local Window = Loader:CreateWindow({ Name = "My UI" }) local Main = Window:CreateTab("Main") Main:CreateButton({ Name = "Test", Callback = function() Window:Notify({ Title = "Button", Content = "You clicked it", Duration = 3 }) end }) 🧩 How your library works internally Your scripts interact like this: Script ↓ require Loader Module ↓ Creates Window ↓ Creates Tabs ↓ Creates UI elements So your code controls the UI through the module. ⚡ Pro tip When making your own UI library, always structure it like this: ReplicatedStorage Loader
