How it Started

HyperMenu was created during a long summer of Roblox development. We were building a massive RPG and realized that the default GUI systems were too slow for high-stakes gameplay.

We wanted a menu that felt "instant." After months of testing different ModuleScripts and UI layouts, HyperMenu was born—a lightweight, fast, and easy-to-use menu for every Roblox developer.

Studio Setup

Follow these steps to get HyperMenu working in your game:

  1. Download the HyperMenu Model from the Creator Store.
  2. Drag the HyperMenuModule into ReplicatedStorage.
  3. Create a LocalScript in StarterPlayerScripts.

Initialize Script:

-- LocalScript inside StarterPlayerScripts
local HyperMenu = require(game.ReplicatedStorage:WaitForChild("HyperMenuModule"))

HyperMenu:Init({
    Theme = "Dark",
    DebugMode = false
})

Default Keybinds

Action Key Function
Toggle GUI F4 Opens or closes the main menu.
Close All ESC Instantly exits all active menus.
Confirm Select Enter Triggers the selected button action.

Module Config

Example table for your Configuration Module:

local Config = {
    ["MenuName"] = "HyperMenu V3",
    ["PrimaryColor"] = Color3.fromRGB(0, 162, 255),
    ["TweenSpeed"] = 0.2,
    ["Keybind"] = Enum.KeyCode.F4
}

return Config
